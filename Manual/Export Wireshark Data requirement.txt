To use wireshark data as dataset try exporting wireshark capture and filter the results as .csv file with requirement parameters below :
'No.', 'Time', 'Source IP', 'Source port', 'Destination IP', 'Destination Port', 'Protocol', 'Packet Length', 'Data Length', 'Data', 'Data Text', 'Info'

parameters to be used in the program :
- Source IP
- Source port
- Destination IP
- Destination Port
- Protocol
- Packet Length
- Data Length
- Data
- Data Text
- Info
